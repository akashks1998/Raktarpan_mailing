# Backend
Backend for Kronos.
## Formats
### Registration form
``` json
{
"name":"Aaa",
"username": "user1"
"email":"aaa@gmail.com",
"password":"password"
}
```

### Login form
``` json
{
"username": "user1",
"password":"password"
}
```

``` json
{
"username": "user1",
"password":"password",
"type":"fixed",
"start":"31/10/18",
"start_time":"08:00am",
"end":"31/10/18",
"end_time":"10:00am",

}
```
